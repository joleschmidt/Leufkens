# Project Brief

## Overview
Website for "Gemeinschaftspraxis Leufkens & Dr. Ismer-Leufkens", a general practitioner and diabetology practice in Sinn. The site aims to provide essential information to patients in a professional, accessible, and friendly manner.

## Core Requirements
- **Information Hub**: Clear display of opening hours, contact details, and location.
- **Service Overview**: Description of medical services (General medicine, Diabetology).
- **Team Introduction**: Introduce the doctors and practice team.
- **Legal Compliance**: Impressum and Datenschutz pages.

## Design Philosophy
- **Aesthetic**: Professional, calming, and clean using a Teal and Stone color palette.
- **UX**: Fast loading, mobile-responsive, and easy to navigate.
- **Tech**: Modern static site using Astro for performance.

## Current Status
Project is in active development with a solid structural foundation in place.
